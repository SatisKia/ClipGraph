ClipGraph Version 4.2.2

はじめに

    「ClipGraph」をダウンロードして下さいまして有難うございます。

    「ClipGraph」は関数グラフ表示ツールです。CLIP用の内部関数、外部関数を実行
    し、グラフ表示できます。

    CLIP言語は、オリジナルの言語です。仕様につきましては、
        http://www5d.biglobe.ne.jp/~satis/clip/language.html
    を参照してください。

インストール

    解凍して生成された"ClipGraph-win32-x64"フォルダを適当な場所にコピーしてくだ
    さい。

アンインストール

    "ClipGraph-win32-x64"フォルダを削除してください。

「ClipGraph」の実行

    "ClipGraph-win32-x64"フォルダ内の"ClipGraph.exe"を実行してください。
